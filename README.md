  ## Usage

  ğŸ’½ To start using Weather-Dashboard, just type your desired town in the search bar. Weather-Dashboard will automatically create a main card to show current day weather with info for Temperature, Wind and Humidity; it will also create 5 cards showing the weather for the following 5 days!
  And if you need to look in a town you previously searched, just click on the right button in your Search History!
  This Web-App uses OpenWeather API.
